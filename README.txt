Reusable Local Business Website Template

How to use:
1. Open index.html in your browser.
2. To create a website for another shop, edit only config.js.
3. Put new client images inside the assets folder.
4. Update products, phone number, WhatsApp number, address and colors in config.js.
5. Upload the folder to Netlify when ready.

Important:
- Enquiry form sends details to WhatsApp.
- Product enquiry buttons also open WhatsApp.
- Google Map is loaded automatically from mapQuery in config.js.
- No backend is required.
